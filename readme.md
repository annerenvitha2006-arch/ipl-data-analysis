# IPL Data Analysis

## Project Overview

This project analyzes Indian Premier League (IPL) match data using Python, Pandas, and Matplotlib. The goal is to explore team performance, toss impact, match locations, and venue statistics through data analysis and visualization.

## Dataset

The dataset used in this project contains IPL match information, including teams, winners, venues, cities, toss results, and match outcomes.

## Tools and Libraries

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* Git
* GitHub

## Analysis Performed

1. Team Wins Analysis

   * Identified the most successful IPL teams based on total match wins.

2. Toss Impact Analysis

   * Examined whether winning the toss increases the probability of winning the match.

3. City-wise Match Analysis

   * Analyzed which cities hosted the highest number of IPL matches.

4. Venue Analysis

   * Identified the most frequently used IPL stadiums and venues.

5. Toss Decision Analysis

   * Studied how often teams chose to bat first or field first after winning the toss.

## Visualizations

The project includes visualizations for:

* Top teams by wins
* Toss impact on match results
* Top cities hosting IPL matches
* Top IPL venues
* Toss decision preferences

## Key Findings

* Some teams consistently achieved higher win counts than others.
* Winning the toss does not guarantee winning the match.
* A small number of cities host a large share of IPL matches.
* Certain venues are used much more frequently than others.
* Teams show clear preferences in toss decisions depending on conditions.
